# MQTT采样器
## 构造MQTT采样器
目前构造mqtt采样器需要基于jmeter, 具体构造过程如下：
### jmeter安装http2插件
- 下载mqtt-jmeter插件依赖，下载地址：https://github.com/emqx/mqtt-jmeter/releases
- 将下载的依赖放到 jmeter的 lib/ext目录下，重启后便可创建mqtt采样器
